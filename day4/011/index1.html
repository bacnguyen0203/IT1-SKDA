        Bước 1 : khai báo tên font mới và đường dẫn cho font đó bằng việc sử dụng
        cú pháp @font-face

        khai báo tên font mới
         @font-face {
            font-family: "Lato123";
            src: url("fonts/Lato/Lato-Regular.ttf");
        }
        trong đó font-family: "Lato123"; ( khai báo tên font )
        src khai báo đường dẫn của font
